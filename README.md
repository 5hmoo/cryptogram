# Simple Cryptogram Generator

A high-contrast, privacy-focused utility for creating printable cryptogram puzzles.
Generates a simple cryptogram from your input, which can then be printed, saved as an image, or played.

## About this Project
This tool was built to provide a simple, clean way to generate cryptograms for print or digital sharing. Unlike many online generators, this tool runs entirely in your browser—meaning the text you enter is never sent to a server or stored. It is designed with accessibility and ease of use in mind.

## Features
- **Wingdings Mode:** Toggle between standard letters and symbols for a classic "decoder" feel.
- **High-Contrast Design:** Built with a 7:1 contrast ratio to ensure readability for all users.
- **Privacy First:** No data leaves your computer.
- **Export Options:** - **Print Ready:** Includes a clean layout for the puzzle and an automatic Answer Key with a decoder legend on a separate page.
  - **Save as Image:** Export your puzzles as PNG files for easy sharing.

## How to use
1. Enter your secret phrase in the editor.
2. Choose whether you want standard letters or Wingdings symbols.
3. Click **Generate** to create the puzzle and hide the original phrase.
4. Solve the puzzle on-screen or use the **Print** or **Save as Image** buttons to share it.
